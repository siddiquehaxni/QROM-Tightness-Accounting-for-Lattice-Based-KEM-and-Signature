# Explicit QROM Tightness Accounting for Lattice-Based KEM and Signature Templates

## Overview

This repository accompanies the research paper:

**"Explicit QROM Tightness Accounting for Lattice-Based KEM and Signature Templates"**

The repository provides reproducibility documentation, formal verification obligations, estimator configurations, supplementary materials, and supporting artifacts related to the analysis of Quantum Random Oracle Model (QROM) security reductions for lattice-based cryptographic schemes.

## Research Scope

The work studies:

- Lattice-Based Key Encapsulation Mechanisms (KEMs)
- Fiat-Shamir-with-Aborts Signature Schemes
- Quantum Random Oracle Model (QROM)
- Tight Security Reductions
- ML-KEM (Kyber-like)
- ML-DSA (Dilithium-like)
- Module Learning With Errors (MLWE)
- SelfTargetMSIS
- Core-SVP Security Analysis
